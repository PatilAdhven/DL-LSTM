## 3. LSTM Mathematical Model Explanation (Mandatory)
Understanding Long Short-Term Memory (LSTM) operations:
- **Input Gate**: Decides what new information is going to be stored in the cell state. It uses a sigmoid layer to decide which values to update and a tanh layer to create a vector of new candidate values.
- **Forget Gate**: Decides what information gets thrown away from the cell state. A sigmoid layer looks at the previous hidden state and the current input, outputting a value between 0 (completely get rid of) and 1 (keep entirely).
- **Cell State**: The core "memory" of the LSTM. It is updated by multiplying the old cell state by the output of the forget gate, and then adding the new candidate values determined by the input gate.
- **Output Gate**: Determines what the next hidden state will be, which contains information on previous inputs. A sigmoid layer evaluates what parts of the cell state make it to output, which is then passed through a tanh layer and multiplied together.
- **Sequence Learning Flow**: By retaining this constant cell state mechanism alongside the evolving hidden state context, the LSTM preserves dependency across long text sequences—solving the vanishing gradient problem inherent in standard RNNs.

## 5. Usage & Deployment Instructions

### Local Execution
1. Install dependencies: `pip install -r requirements.txt`
2. Train the model and generate tokenizers/weights (creates `.h5` and `.pkl`):
   ```bash
   python train_model.py
   ```
3. Run the FastAPI server: 
   ```bash
   uvicorn main:app --reload
   ```
4. Test the API through the FastAPI Interactive Dashboard: Go to [http://localhost:8000/docs](http://localhost:8000/docs)
5. Under `POST /predict`, try executing a payload:
   ```json
   {
       "seed_text": "deep learning is",
       "top_n": 3
   }
   ```
